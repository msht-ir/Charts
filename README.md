Draw column charts  in Excel (single series / dual series).
Use Sheet1 workSheet in your workBook.
Copy codes in Module1 of your Excel workBook.

Chart data-blocks:
Row-1
A1: chartName 
B1: Y-Axis-Title
Row-2
A2: chartType (1: single series chart, 2: dual series chart)
B2: X-Axis-Category 1
C2: Data
D2: letters for onTop of SD-bars
E2: SD values
Row-3
B3: X-Axis-Category 2
C3: Data
D3: letters for onTop of SD-bars
E3: SD values
Row-4 ...
